RESTful Routes Table

A table of all 7 RESTful Routes!

Name Path HTTP Purpose Mongoose Method
Index /cats GET Lists all the cats Cat.find()
New /cats/new GET Shows new cat form N/A
Create /cats POST Creates a new cat, then redirects somewhere Cat.create()
Show /cats/:id GET Shows info about one specific cat Cat.findById()
Edit /cats/:id/edit GET Shows edit form for one spefic cat Cat.findById()
Update /cats/:id PUT (requires override) Update a particular cat, and then redirects somewhere Cat.findByIdAndUdate()
Delete /cats/:id DELETE (requires override) Deletes a particular cat, and then redirects somewhere Cat.findByIdAndRemove()
